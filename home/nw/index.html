HJELP

For å gjøre nedlasting og distribusjon av EasyPHP lettere, er bare nødvendige elementer fra Apache, PHP og MySQL tatt inn i EasyPHP.

Før du åpner "Administrasjon" eller "Lokal Web" fra EasyPHP-menyen [2], kontrollér at Easyphp.exe er startet (f.eks.: Start>Programmer>EasyPHP>EasyPHP) og at serverne fungerer (dobbelt-klikk på EasyPHP-ikonet i systemstatusfeltet ved siden av klokken [1]: Apache og MySQL må ha grønt status).

Administrasjon

Høyre-klikk på EasyPHP-ikonet [1] og velg "Administrasjon"[2]. På denne siden kan du bl.a. administrere dine databaser og aliaser.

Lokal Web

Høyre-klikk på EasyPHP-ikonet [1] og velg "Lokal Web" [2] for å se på dine filer (jf. "Innledning til EasyPHP").
