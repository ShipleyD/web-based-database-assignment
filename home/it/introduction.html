INTRODUZIONE

Installare EasyPHP

  • Scaricare EasyPHP dal sito www.easyphp.org
  • Doppio Click sul file scaricato
  • Selezionare il percorso di installazione e seguire la procedura guidata

Avviare EasyPHP

Non si può parlare propriamente di avvio di EasyPHP, in realtà vengono avviati i server di Apache e MySQL. In fase di installazione, viene creato un collegamento a EasyPHP nella cartella "Start>Programmi>EasyPHP". Una volta avviato EasyPHP, un'icona si posizionerà a fianco all'orologio [1].

Un click col tasto destro consente l'accesso ai menu [2]:
  • Help : guida a EasyPHP
  • File Log : archivia gli errori generati da Apache, MySQL e EasyPHP
  • Configurazione : permette l'accesso ai vari parametri di configurazione
  • Esplora file: apre la cartella "www" nel browser predefinito di Windows
  • Amministrazione : apre la pagina di amministrazione
  • Sito Locale: accede sito locale
  • Riavvia : riavvia Apache e MySQL
  • Avvia/Ferma : avvia/ferma Apache e MySQL
  • Esci : chiude EasyPHP

Utilizzare la cartella "www" o gli alias

Affinchè le vostre pagine php siano interpretate, è obbligatorio inserire i vostri file nella cartella "www" o in un alias appositamente creato. Per visualizzare le pagine create è poi sufficiente aprire "Sito Locale" o accedere al vostro alias dalla pagina di "Amministrazione".

Le vostre prime pagine in PHP

Esistono molti editor per scrivere pagine PHP, specifici per il linguaggio e non (editor html, con colorazione della sintassi...). In questo esempio useremo un editor di testo semplice.

Create un nuovo file
Digitate la struttura di una pagina HTML vuota :

<html>
<head>
<title>La mia prima pagina in PHP</title>
</head>
<body>

</body>
</html>
L'esempio consiste nel visualizzare la data corrente. Il codice PHP si integra direttamente al codice HTML. L'intero codice deve essere scritto tra i tag.
<html>
<head>
<title>La mia prima pagina in PHP</title>
</head>
<body>

Data corrente : <?php print (Date("l F d, Y")); ?>

</body>
</html>

Salvataggio della pagina.
Creare una nuova cartella sotto la cartella "www" (o in un alias). Salvate la vostra prima pagina PHP dando una delle estensioni seguenti: php (consigliata), php3, php4.
Questa non è una regola universale, ma corrisponde alla configurazione di EasyPHP. Potrebbe essere necessario cambiare le estensioni dei file nel caso sceglieste un servizio di hosting che usa una configurazione differente.
