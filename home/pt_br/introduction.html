INTRODUÇÃO

Instalar o EasyPHP

  • Baixar o EasyPHP do site www.easyphp.org
  • Duplo clique sobre o arquivo executável (exe)
  • Selecione o local de instalação e siga as instruções

Iniciar o EasyPHP

O EasyPHP não inicia como os demais programas. Trata-se, na verdade, de um dispositivo que inicia os servidores Apache e MySQL. Durante a instalação, é criado um atalho para o EasyPHP em "Iniciar/Programas/EasyPHP". Após iniciado o EasyPHP, surge um ícone na bandeja, ao lado do relógio do Windows [1].

Clique direito sobre o ícone do EasyPHP para acessar as diferentes opções de menu [2]:
  • Ajuda : Ajuda do EasyPHP
  • Arquivos de Log : apresenta os relatórios gerados pelo Apache, MySQL e EasyPHP
  • Configuração: permite abrir as diferentes ferramentas de configuração
  • Explorar : abre a pasta "www" do Apache no Windows Explorer
  • Administração: abre a página de administração
  • Web local: abre a web local (locahost)
  • Reiniciar: reinicia o Apache e o MySQL
  • Iniciar/parar: inicia/pára o Apache e o MySQL
  • Sair: encerra o EasyPHP

Use a pasta "www" ou os alias

Para que as suas páginas PHP sejam interpretadas, é necessário colocar suas arquivos dentro da pasta "www" ou em uma pasta que corresponda a um alias criado. Para visualizar suas páginas, basta abrir a "Web local" ou acessar seus alias através da página de administração.

Suas primeiras páginas em PHP

Existem inúmeras formas de programar em PHP e diversos programas de edição. Neste exemplo, será empregado um simples editor de texto.

Abra um novo arquivo
Crie a estrutura de uma página HTML:

<html>
<head>
<title>Minha primeira página em PHP</title>
</head>
<body>

</body>
</html>
O exemplo consiste em criar a data atual. O código PHP integra-se diretamente no HTML, começando com “”.
<html>
<head>
<title>Minha primeira página em PHP</title>
</head>
<body>

Data atual : <?php print (Date("l F d, Y")); ?>

</body>
</html>

Salvar página.
Crie uma nova pasta em "www" (ou em um alias). Salve sua primeira página em PHP com uma das seguintes extensões: php (recomendado), php3, php4.
Não se trata de uma regra absoluta, mas corresponde à configuração do EasyPHP. Caso caso pretenda hospedar suas páginas em um servidor cuja configuração seja diferente, poderá ser necessário modificar estas extensões.
