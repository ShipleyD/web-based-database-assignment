INTRODUCCIÓN

Instalar EasyPHP

  • Descargar EasyPHP en la Web www.easyphp.org
  • Haga Doble clic sobre el programa ejecutable que ha descargado
  • Seleccionar el directorio de instalación y seguir los pasos.

Lanzar EasyPHP

Nos se puede decir que se ha arrancado EasyPHP, en realidad, se trata del arranque del servidor Apache y de la base de datos MySQL. Durante la instalación, un enlace hacia EasyPHP se crea en el directorio "Inicio/Programas/EasyPHP". Una vez EasyPHP lanzado, un icono se añade al lado del reloj [1].

Un clic derecho, permite de acceder al los diferentes menús [2]:
  • Ayuda : ayuda de EasyPHP
  • Ficheros Log : envían los errores generados por Apache, MySQL e EasyPHP
  • Configuración : da acceso a las herramientas de configuración
  • Explorer : abre el directorio "www" por medio del explorador de Windows
  • Administración : abre la página de l'administración
  • Web local : abre la Web local
  • Volver a arrancar: Arranca de nuevo Apache y MySQL
  • Arrancar/Parar : Arranca/Parar Apache y MySQL
  • Salir : cierra EasyPHP

Utilizar el directorio "www" o los alias

Para que sus páginas PHP sean interpretadas, es imperativo añadir sus ficheros en el directorio "www" o en un alias que usted halla creado. Para visualizar sus páginas, le basta entonces con abrir la "Web local" o ir a los alias por medio de la página de "Administración".

Sus primeras páginas en PHP

Hay tantas formas de programar en PHP como de editores epecializados (editores html, con coloración de sintaxis, escritura semiautomática ...). En este ejemplo vamos a usar un editor de texto simple.

Abra un nuevo fichero
Escriba la estructura de una página HTML blanca:

<html>
<head>
<title>Mi primera página en PHP</title>
</head>
<body>

</body>
</html>
El ejemplo consiste en mostrar la fecha actual. El código PHP se integra directamente en el código HTML y comienza siempre por .
<html>
<head>
<title>Mi primera página en PHP</title>
</head>
<body>

Fecha actual : <?php print (Date("l F d, Y")); ?>

</body>
</html>

Salvar la página.
Crear un nuevo directorio en el directorio "www" (o en un alias). Salvar su primera página en PHP dandole una de las extensiones siguientes : php (aconsejado), php3, php4.
Esto no es una regla absoluta, pero corresponde a la configuración de EasyPHP. Le sera quizas necesario si elige de alojar sus páginas en un alojamiento de configuración diferente, de cambiar estas extensiones.

Visualización del resultado.
* A NO HACER : ir al directorio "www" (o al directorio de un alias) y despues al directorio correspondiente de su proyecto y doble clic en su página de ejemplo. Obtendrá siempre un error.
* A HACER : lanzar EasyPHP, abrir la "Web local", seleccionar vuestro directorio de trabajo (o vuestro alias) y después clic sobre "date.php". Obtendrá entonces una página que le enseñara la fecha actual. Por ejemplo : "Fecha actual: Wednesday March 22, 2006".
